# 零域 (Zero Domain) - Web Storefront 🌌

> **"将无限的大模型算力，封装于零域之内。"** > 
> 本项目是 **零域 (Zero Domain)** 品牌的官方独立站前端展示层。专注为硬核开发者与极客提供本地 AI 算力硬件的极致购物体验。

## 📖 项目简介

这是一个基于 **Vue 3 (Composition API)** 构建的现代化、响应式无头电商（Headless Commerce）前端项目。

区别于传统的电竞硬件商城，本项目在 UI 设计上采用了极简的**暗黑代码风 (Dark/Hacker Mode)**，去除了冗余的促销干扰，专注于“显存叠加”、“统一内存”等核心算力参数的交互式展示。本项目主要用于前端技术栈的综合演练，不涉及真实的后端业务逻辑，数据均由本地 Mock 服务驱动。

## 🎯 核心技术栈

* **核心框架：** Vue 3 (Setup 语法糖) + Vite
* **状态管理：** Pinia (用于跨页面购物车管理与复杂 SKU 联动)
* **路由控制：** Vue Router 4
* **样式方案：** CSS 3 / Flex / Grid (专注于精细化布局与极客感色彩管理)
* **数据模拟：** JSON Server (提供本地 RESTful API 支持)

## ✨ 核心亮点与练手模块

1. **沉浸式单品详情页 (PDP)：**
   * 实现复杂的动态 SKU 配置器（支持单卡/多卡并联、统一内存容量动态计算）。
   * 随配置动态切换的硬件渲染图与总价计算。
2. **极简购物车抽屉：**
   * 全局状态接管，平滑的滑出动效与数字步进器。
3. **随动交互与滚动视差：**
   * 监听页面滚动轨迹，实现具有通透感的吸顶导航栏与元素的优雅渐入（Scroll Reveal）。

## 📦 零域 (Zero Domain) 虚拟产品线设定

本项目内置以下三款核心产品的交互数据：
* **Z-Node (入门算力节点)：** 专注轻量化本地推理，极简配置（练习基础表单绑定）。
* **Z-Core Pro (桌面级算力核心)：** 主力销售机型，支持双显卡并联与海量内存定制（练习复杂状态联动与定价算法）。
* **Z-Matrix (企业级统一内存集群)：** 模块化阵列方案（练习动态组件生成）。
